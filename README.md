# Smart Library Manager

A Java-based Library Management System developed using Java Swing and MySQL.  
This project helps manage books, students, issue/return records, and library operations through a graphical user interface.

---

## Features

- User Login System
- Add / Update / Delete Books
- Manage Student Records
- Issue Books
- Return Books
- Search Functionality
- MySQL Database Integration
- Simple and User-Friendly GUI
- JDBC Connectivity

---

## Technologies Used

- Java
- Java Swing
- MySQL
- JDBC
- Eclipse IDE
